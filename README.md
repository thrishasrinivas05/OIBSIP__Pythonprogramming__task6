A simple Weather Application built using HTML, CSS, and JavaScript that fetches real-time weather information using the OpenWeatherMap API.
Users can enter a city name to view the current temperature, humidity, and weather condition.

Features:
 Search weather by city name
 Displays temperature in Celsius
 Shows humidity level
 Shows weather condition
 Handles invalid city names
 Simple and clean user interface

Technologies Used:
 HTML – Structure of the web page
 CSS – Styling and layout
 JavaScript – API request and dynamic data display
 OpenWeatherMap API – Fetching real-time weather data
